Style your Google Maps in Events Manager! Select Map Type, Zoom Level, Controls and more.

== Description ==
Finally you can style the Google Maps on your event pages within Events Manager the way *YOU* like them!
- Set the Zoom Level
- Select the Map Type
- Show or hide Footer texts and more!
This add-on needs the original **Events Manager** plugin to be installed and activated.

**NOTE**
Some changes might require you empty your browser's cache.

== Installation ==
Download from the Wordpress Repository by searching for "Events Manager Maps Styling" and simply activate.
You can also download the zip-file and upload the unzipped folder using FTP to /wp-content/plugins.

After activation, go to Dashboard -> EM Add-ons -> Maps Styling to customize your settings.

**NOTE**
Some changes might require you empty your browser's cache.
